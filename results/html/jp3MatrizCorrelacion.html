5. Graficando

Una matriz de correlacin es una herramienta estadstica que muestra la intensidad y la direccin de la relacin entre dos o ms variables. Se utiliza mucho en campos como las finanzas, la economa, la psicologa y la biologa, porque ayuda a entender cmo se relacionan entre s distintas cosas.

In[]:
#Creando la mtriz de correlacion
print("Matriz de Correlacion:\n")
matriz_correlacion=data.corr().round(2)

5.1 Mapa de Calor en Seaborn con heatmap (antes)

La funcin heatmap permite crear mapas de calor o grficos de correlacin en Python con seaborn. Puedes pasar un conjunto de datos 2D o un data frame de pandas. En caso de utilizar un data frame de pandas los nombres de las filas y las columnas sern usados como etiquetas para los ejes.

In[]:
#ruta = "../results/dataset_penguins_clean.csv"
# Diagrama de Calor
plt.figure(figsize=(10, 8))
sns.heatmap(data=matriz_correlacion,  # pasandoles los datos CSV con el nuevo DF
            annot=True,  # Mostrar los datos dentos de los cuadros
            square=True,  # Celdas cuadrdas
            annot_kws={'size': 12},  # Tamao de los datos en los cuadros
            linecolor='white',  # Color del borde de los cuadros
            linewidths=0.75)  # Grosor del borde de los cuadros
plt.show()
